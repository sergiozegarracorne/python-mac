Buena pregunta, aclaro los dos conceptos porque se mezclan:

## "Reentrenar" vs "agregar frases" — no es lo mismo

Cada vez que apretás "Entrenar modelo", Rasa **empieza de cero** y solo mira lo que hay en ese momento en `nlu.yml`. No hay memoria acumulada entre entrenamientos — si volvés a entrenar sin cambiar nada, el modelo nuevo va a ser prácticamente idéntico al anterior (con mínimas diferencias por aleatoriedad). **Reentrenar sin agregar frases no lo hace más inteligente.**

Lo que sí mejora la interpretación es **agregar frases nuevas y variadas** a `nlu.yml` y ahí sí reentrenar. Ahí el modelo tiene más patrones de los que aprender.

## Qué significa "epochs: 100" en el config

Eso es cuántas vueltas da el algoritmo sobre **los mismos ejemplos** durante un solo entrenamiento (viste el log `Epochs: 100/100`). Subir ese número no agrega conocimiento nuevo, solo ajusta más finamente los pesos con los datos que ya tiene — pasado cierto punto no mejora más, e incluso puede empezar a "memorizar" en vez de generalizar (overfitting). Con ~15-30 ejemplos por intent, 100 epochs ya es más que suficiente (por eso vimos accuracy 100% rápido).

## Cómo se afina la interpretación en serio

1. **Variedad real, no repetición** — en vez de 50 frases casi iguales, mejor 15-20 bien distintas (formal, informal, con errores de tipeo, cortada, con "onda", etc.)
2. **Ejemplos "difíciles" a propósito** — frases parecidas pero de OTRO intent, para afilar el límite entre ambos. Esto fue justo lo que hicimos con "cuánto debe" (deudas) vs los demás, y con `informe_nis` vs `consulta_estado_nis` — esa técnica es la que más sube la precisión real.
3. **Balancear cantidad por intent** — si un intent tiene 5 ejemplos y otro 50, el modelo se sesga hacia el que tiene más. Conviene una cantidad pareja.
4. **Manejar lo desconocido** — hoy si le mandás una frase rarísima, el modelo igual "fuerza" una respuesta con la intención que más se parezca (aunque sea con poca confianza). Se puede agregar un umbral de confianza + intent `nlu_fallback`, para que diga "no entendí" en vez de inventar una intención. Esto es clave en producción real.
5. **El feedback más valioso: producción real** — cuando esto ya esté funcionando con usuarios reales, las frases que el modelo clasificó mal (o dudó) son oro: las agregás a `nlu.yml` con la etiqueta correcta y reentrenás. Así se afina de verdad con el tiempo — no adivinando ejemplos, sino con los casos reales que fallan.

def remove_accents_simple(text):
    replacements = {
        "á": "a",
        "é": "e",
        "í": "i",
        "ó": "o",
        "ú": "u",
        "Á": "A",
        "É": "E",
        "Í": "I",
        "Ó": "O",
        "Ú": "U",
    }

    for original, replacement in replacements.items():
        text = text.replace(original, replacement)

    return text


def add_typo(text):
    candidates = [
        (a, b)
        for a, b in TYPO_REPLACEMENTS
        if a in text.lower()
    ]

    if not candidates:
        return text

    original, replacement = random.choice(candidates)

    return re.sub(
        re.escape(original),
        replacement,
        text,
        count=1,
        flags=re.IGNORECASE,
    )


def remove_random_vowel(text):
    """
    Simula errores:
        quiero -> quier
        consulta -> conslta

    Nunca modifica números ni códigos.
    """
    words = text.split()

    candidates = [
        i
        for i, word in enumerate(words)
        if len(word) >= 6
        and not re.search(r"\d", word)
    ]

    if not candidates:
        return text

    index = random.choice(candidates)
    word = words[index]

    vowel_positions = [
        i
        for i, char in enumerate(word)
        if char.lower() in "aeiou"
        and i > 0
    ]

    if vowel_positions:
        pos = random.choice(vowel_positions)
        words[index] = word[:pos] + word[pos + 1:]

    return " ".join(words)


def mutate_case(text):
    option = random.choice([
        "lower",
        "lower",
        "lower",
        "upper",
        "capitalize",
    ])

    if option == "upper":
        return text.upper()

    if option == "capitalize":
        return text.capitalize()

    return text.lower()


def mutate_punctuation(text):
    option = random.choice([
        "none",
        "none",
        "question",
        "double_question",
        "dots",
        "comma",
    ])

    if option == "question":
        return text + "?"

    if option == "double_question":
        return text + "??"

    if option == "dots":
        return text + "..."

    if option == "comma":
        return text.replace(" ", ", ", 1)

    return text


def apply_noise(text):
    """
    Aplica ruido controlado y devuelve:
        texto_modificado, tipo_de_ruido
    """

    noise = random.choice([
        "limpio",
        "limpio",
        "jerga",
        "typo",
        "sin_vocal",
        "mayusculas",
        "puntuacion",
        "combinado",
    ])

    if noise == "limpio":
        return text, noise

    if noise == "jerga":
        text = random.choice(FILLERS_START) + text
        text += random.choice(FILLERS_END)

    elif noise == "typo":
        text = add_typo(text)

    elif noise == "sin_vocal":
        text = remove_random_vowel(text)

    elif noise == "mayusculas":
        text = mutate_case(text)

    elif noise == "puntuacion":
        text = mutate_punctuation(text)

    elif noise == "combinado":
        text = random.choice(FILLERS_START) + text
        text += random.choice(FILLERS_END)

        if random.random() < 0.7:
            text = add_typo(text)

        if random.random() < 0.3:
            text = remove_random_vowel(text)

        text = mutate_case(text)
        text = mutate_punctuation(text)

    return text.strip(), noise

def build_case(intent):
    if intent in TEMPLATES:

        template = random.choice(TEMPLATES[intent])

        if intent == "consulta_nis_por_medidor":
            value = random_medidor()

            assert is_valid_medidor(value), (
                f"Medidor inválido generado: {value}"
            )

            text = template.format(medidor=value)

        else:
            value = random_nis()

            assert is_valid_nis(value), (
                f"NIS inválido generado: {value}"
            )

            text = template.format(nis=value)

        text, noise = apply_noise(text)

        return {
            "text": text,
            "expected_intent": intent,
            "expected_value": value,
            "noise": noise,
        }

    text = random.choice(EXTRA[intent])
    text, noise = apply_noise(text)

    return {
        "text": text,
        "expected_intent": intent,
        "expected_value": None,
        "noise": noise,
    }


def build_cases(total):
    intents = list(TEMPLATES.keys()) + list(EXTRA.keys())

    cases = []

    # Distribución equilibrada
    while len(cases) < total:
        for intent in intents:

            if len(cases) >= total:
                break

            cases.append(build_case(intent))

    random.shuffle(cases)

    return cases

def extract_entity_values(entities, expected_intent):

    if expected_intent == "consulta_nis_por_medidor":

def print_confusion_matrix(confusion, intents):

    log("")
    log("MATRIZ DE CONFUSIÓN")
    log("-" * 100)

    width = 25

    header = "ESPERADO".ljust(width)

    for intent in intents:
        header += intent[:15].center(17)

    log(header)

    for expected in intents:

        row = expected[:24].ljust(width)

        for predicted in intents:
            value = confusion[expected][predicted]
            row += str(value).center(17)

        log(row)
